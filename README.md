# Car Booking System (CLI)

## Overview
This is a simple **Car Booking Command-Line Interface (CLI) system** designed for a car company admin.  
The goal of this initial version is to allow an admin to **book cars** and **view users** using a clean, array-based implementation.

This project focuses on core Java OOP concepts and avoids advanced tools or frameworks to maintain simplicity.
