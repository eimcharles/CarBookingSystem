## Features </br>
- Single Sign-On / Sign-up (SSO)
- Role-Based Access Control
- Asynchronous Email Notifications
- Integrated Payment Processing


## Learning Outcomes: </br>
- Spring Boot to manage project's configurations.
- Spring Data JPA with MySQL database for persistent storage and retrieval of application data.
- Spring Security Core User authentication and single sign on.
- Spring Boot Mail Starter for notifications.
- Streams for functional programming.
- Generics for type-safe API response mapping.
- Mocking using Mockito for testing component interactions.
- Java Test Containers to for isolated database environments for integration tests
- Abstraction / Polymorphism to define clear interfaces for components.
